1. git installation check or version check
    git --version

2. user name set on git globally(for all files)
    git config --global user.name "your-name"

3. user email set on git globally(for all repositories)
    git config --global user.email "your-email.@example.com"

4. user name set on git (for a specific repository)
    git config  user.name "your-name"

5. user email set on git (for a specific repository)
    git config user.email "your-email.@example.com"

6. git initialization on a directory
    git init

7. check current stage of git in a directory
    git status

8. to stage a file for the next commit
    git add fileName.extension

9. to stage all files in a directory for the next commit
    git add .
